# Phishing Awareness Game

## About This Project
This project was focused on creating a cybersecurity awareness game. My team and I decided to create a game which was
interactive and entertaining for non technical users as well. The aim of this game was to educate users to be mindful 
when accessing emailsdue to the increasing cyber threats and scams, especially phishing.

This project was built to benefit the community as staff asper our rubrics requirements. The project was initiated by
collecting data on phishing attacks to inform users about cyber threats and its severity.

## How The Game Works
To make it entertaining for users, we included elements like character selection page. Following this,
the game initiates with the player running on a straight track and to clear the obstacles they have to choose
the right email from two boxes — one contains the correct email and one contains the fraudulent scam email.

- If the player chooses the right email, they move to the next level
- If the wrong email is chosen, a screen appears notifying them why the email was a scam and what the
-  hidden hints are that one should consider.
- The right email leads the player to an advanced level

Our target was to cover smishing, vishing, and phishing. However,
due to time limitations, we focused on phishing only.
